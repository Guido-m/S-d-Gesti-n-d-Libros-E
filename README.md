# Sistema de Gestión de Libros Electrónicos

Proyecto académico desarrollado en Go (Golang) que permite registrar, consultar, actualizar y eliminar libros electrónicos, aplicando principios de programación funcional.

## Tecnologías
- Lenguaje: Go (Golang)
- Paradigma: Programación funcional
- Entorno de desarrollo: GoLand
- Control de versiones: Git y GitHub
